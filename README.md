# Sadist 2 - Ostatni krzyk | Sadist 2 - Final Scream

Sadist 2 — a classic DOS game from 2000.
The successor to Sadist.

Created by Grzegorz Drozd a long time ago...

# Introduction

Everything you should know about Sadist 2 is included in the file [sadist2-manual-en](sadist2-manual-en.md). Please read it before starting the game.

Version history is in [history-en](history-en.md).

# Website

The Sadist 2 website is available at https://gadz.pl/sadist-2.
You can download a working copy of the game there, as well as videos, screenshots, descriptions, and more.

A gameplay video is here: https://www.youtube.com/watch?v=MA6PZX7lKpY

# Compilation

Sadist 2 was created in Turbo Pascal 7.0 by Borland International Inc.
The best compiler for the source code is TP or Borland Pascal — they are almost the same. A free Borland Pascal Compiler is available on the internet, so if you need it, you should be able to find it. I cannot distribute it because it is not my software.

All required source files should be included in this repository. The main entry point is `sadist2.pas`. There are also a few additional small tools in other files, but most `.pas` files are units of the Sadist 2 game.

# The code

This is a Polish game. I was creating it when I was young and learning to code, so this code is not top quality. Almost all variable and type names are in Polish, so it may be hard to understand for non-Polish speakers. You'll also see a big mess in the code, ugly formatting, and so on. Sorry — that was my fault. You may believe me, but today I do code reviews for other programmers at work and I don't allow anyone to write code like what's here! BUT! All of this shows how a person learns and develops their skills.

The code will be mostly automatically formatted to be more readable. For now, it is completely the same as it was more than 20 years ago.

## English version of the game

In 2026 I made the English version of Sadist 2. It is also available in this repo. I've translated all visible Polish sentences and captions, as well as all additional text files.
