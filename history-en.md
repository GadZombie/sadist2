## Version 1.07 fajnal — 2006-04-01

+ This is not an April Fools' joke :-)

* This version was made to remove the 'standard set' from the Sadist website. At present, the game should work right after installation, without needing to download other files (of course, alternative character and graphics sets are still available on the website). Two character sets are included with the game package: Zombie and Aliens.
* After installing on Windows, the game adds shortcuts to the Start Menu, including a shortcut to the no-sound version, which may be necessary on Windows XP.
* Small fix to the character list. With more characters, "garbage" would appear on screen.
* Changed SADIST2.TXT file

## Version 1.05 — 2001-07-03

* Small fix with the magnifying glass. When there was little free memory, the program would crash after turning it on.
* Caught characters or crates moved with the cursor to the very bottom of the screen no longer disappear.
* Added character knockback when shooting (while controlling them).
* Improved blood appearance - best visible when detail settings have lots of particles!
* Added options for blood amount and blood fountain toggle.
* The entire main menu is now icons instead of text labels, as before.
* Changed the Sadist logo, which from now on is meant to be its clear, program-associated logo :).
* New weapon: shard thrower.
* Added snarky comments ;).
* New surprise in crates - confetti ;-).
* Character invincibility option.
* Fixed program settings.
* Characters can finally be walled in.
* Fixed crosshair [mouse].
* Added options for rifle, minigun, and shotgun power.
* Mines can also be duds [they don't explode]!
* A controlled character can also walk (not run) while holding the 'V' key.
* The flamethrower can also pour napalm
* History (this file) changed from continuous text to bullet points
* Ability to throw any crates (menu: other attractions) and disable selected types of falling crates

## Version 1.04 — 2001-03-07

* You can now also draw with a smooth color.
* Fixed blood pouring.
* Added option to hide the mouse cursor (F3 key).
* Weapons from individual menus are now remembered separately - just switch menus and the weapon changes to the last one used from that menu [optional].
* Added tabs in the bottom menu.
* You can now navigate the save/load menu with the keyboard too.
* Increased maximum for some detail settings.
* New surprise in crates - mega boom :).

* Lightning is slightly brighter.
* Crates can also be moved with shift - like characters.
* Characters playing with something react more slowly to falling anvils/crates - because they're interested in something else :).
* Option for screen scroll speed with mouse and keyboard [separately].
* Intro changed a bit.
* Fixed a small bug with saving and loading terrain.
* Terrain file format changed slightly. Converting old terrains to the new format will be necessary! Converter available on the website, in the Download section.
* Nothing beats optimization - I managed to reclaim some conventional memory that you can now use for details! :)
* Crates sometimes look different :>.
* Birds sometimes have something to do ;-).
* New weapon!: napalm.
* Detail settings now work on sliders, so they're more convenient to use.
